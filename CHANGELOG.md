# 更新日志

## v1.0.0 (2024-01-19)

### ✨ 新功能

- **主人端功能**
  - 创建饭桌（设置名称、用餐时间、备注等）
  - 菜单管理（添加、编辑、删除菜品）
  - 菜品状态管理（可点、推荐、售罄）
  - 分享饭桌（复制链接）
  - 查看点菜情况（按人查看/按菜品查看）
  - 确认最终菜单
  - 导出菜单（文本格式）
  - 历史饭桌记录

- **客人端功能**
  - 通过链接或二维码进入饭桌
  - 查看菜单（分类浏览）
  - 点菜（增加/减少数量）
  - 查看他人点菜
  - 查看我的点菜统计

- **技术特性**
  - 微信小程序原生框架
  - 微信云开发后端服务
  - 云数据库存储
  - 云函数API
