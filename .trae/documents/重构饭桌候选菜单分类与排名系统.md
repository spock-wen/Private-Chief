# 重构饭桌详情页：全能决策中心与分类菜单系统

## 核心设计原型 (由 ui-ux-pro-max 驱动)
1. **琥珀暖宴视觉系统**：应用玻璃拟态材质与琥珀金弥散阴影，营造专业且温馨的家宴感。
2. **分类排名系统**：按菜品类型分组，展示分类冠军勋章与全场 Top 3 人气榜单。
3. **主人全景透视 (Host Perspective)**：
   - **全景投票矩阵**：主人可一键查看“客人 x 菜品”的完整投票分布。
   - **头像穿透**：菜品卡片展示投票人头像墙，点击可查看该客人的忌口详情。
4. **互动定稿流程 (Selective Locking)**：
   - 主人进入锁定状态前，需手动勾选最终选定的菜品。
   - 侧边栏实时计算“访客口味覆盖率”，辅助主人进行公平定稿。

## 实施路线图
### 1. 数据库层更新 (backend-architect)
- `schema.prisma`: 为 `Table` 模型增加 `finalDishIds` 字段。
- `tables.service.ts`: 实现批量设置最终菜单的 API 逻辑。

### 2. 前端原型重构 (frontend-architect)
- **样式系统**：全量对齐 `ui-ux-pro-max` 推荐的视觉参数。
- **页面重构**：更新 `TableDetail.vue` 模板，实现分类瀑布流与 `SelectionMode`。
- **组件开发**：`VoterMatrix` (透视弹窗), `FinalizeOverlay` (定稿控制台)。

### 3. 联调与校验
- 确保主人与客人身份的严格隔离，保护投票私密性的同时赋予主人决策透明度。
